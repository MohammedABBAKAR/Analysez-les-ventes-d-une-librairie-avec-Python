# Analysez-les-ventes-d-une-librairie-avec-Python
Analysez les ventes d'une librairie avec  Python
## Contexte
* Librairie Lapage : magasins physiques → e-commerce (depuis 2 ans)
* Forte croissance des ventes en ligne
* Données disponibles mais peu exploitées
## Enjeu
* Manque de visibilité globale sur la performance
* Besoin d’aide à la décision stratégique
## Objectif global
Donner une vision claire de la performance e-commerce

## Objectifs spécifiques
* Analyser les indicateurs clés de vente
* Identifier les produits et catégories stratégiques
* Comprendre le comportement des clients
* Aider la direction à la prise de décision
